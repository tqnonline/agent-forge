# agent-forge v1.0.0

**Released: 2026-05-03**

The first stable release of agent-forge — a curated, BSD-3-Clause marketplace
of agent skills, plugins, and slash commands that works across 12 install
targets without per-tool reauthoring.

## Highlights

- **12 install targets** at v1.0: 4 CLIs with native marketplace integration
  (Claude Code, GitHub Copilot CLI, OpenAI Codex CLI, Cursor), 2 with git-URL
  install (Sourcegraph Amp, Google Gemini CLI), 3 with lightweight copy-to-home
  adapters (Kilo Code, OpenCode, Crush), and 3 prompt-tool loaders (Perplexity
  Spaces, ChatGPT custom GPTs, Claude.ai Projects).
- **4 plugins / 46 skills** total: `writing`, `prompts`, `msft-arch`, `pm`.
- **One canonical source** — author plugins in Claude Code SKILL.md format under
  `plugins/<name>/`; translators auto-generate all CLI-specific manifests.
- **`agent-forge` Python CLI** (`pipx install agent-forge`) — unified install,
  update, pin, remove, and doctor across every supported tool.
- **Agent-installable** — paste a single URL into any LLM agent and it follows
  the install guide directly without human intervention.

## Breaking changes from pre-1.0

None — this is the first stable release. Schemas frozen at v1.0:
- `.claude-plugin/marketplace.json`
- `.github/plugin/marketplace.json`
- `.codex-plugin/marketplace.json`
- `.cursor-plugin/marketplace.json`
- Per-plugin `plugin.json` files
- `~/.agent-forge/manifest.json` (`schema_version: 1`)
- `agent-forge` CLI command surface (11 commands)
- The install-URL pattern

After v1.0, breaking changes to any of the above require a v2.0.0 major version bump.

## Install

```bash
# Claude Code
claude plugin marketplace add github:tqnonline/agent-forge
claude plugin install writing

# GitHub Copilot CLI
copilot plugin marketplace add tqnonline/agent-forge

# Codex CLI
codex plugin marketplace add tqnonline/agent-forge

# Amp
amp skill add github:tqnonline/agent-forge

# Gemini CLI
gemini skills install github:tqnonline/agent-forge

# Kilo Code, OpenCode, or Crush
pipx install agent-forge
agent-forge install writing --tier kilocode  # or opencode, crush
```

For Perplexity, ChatGPT GPTs, and Claude.ai Projects — see
[docs/install/_index.md](../install/_index.md).
