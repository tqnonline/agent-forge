# agent-forge v1.0.1

**Released: 2026-05-26**

First release under the `tqnonline` GitHub organization. v1.0.0 was published
from `rahulnakmol/agent-forge` (personal account) and transferred to the org.
v1.0.1 is the first release with `tqnonline/agent-forge` as the canonical
home — embedded in the wheel/sdist defaults, docs, and CI workflows.

## Changes since v1.0.0

- **Canonical owner:** all references migrated from `rahulnakmol/agent-forge`
  to `tqnonline/agent-forge` — install commands, manifest defaults, badge URLs,
  raw-content links, and security advisory paths. GitHub's permanent redirects
  keep v1.0.0 installs working, but new installs should use the canonical org.
- **PyPI distribution name: `tqn-agent-forge`.** PyPI rejected `agent-forge`
  as too similar to the existing `agentforge` project, so the distribution is
  published as `tqn-agent-forge`. The Python module (`import agent_forge`) and
  the CLI command (`agent-forge`) are unchanged. New install: `pipx install
  tqn-agent-forge`.
- **CI fix:** `ci-structural.yml` and `ci-evals.yml` now install the
  `agent_forge` package before running tests (was missing — every CI run since
  Phase 3 had been red on collection errors).
- **Classifier:** `Development Status :: 5 - Production/Stable`.

## Install

```bash
# Claude Code
claude plugin marketplace add github:tqnonline/agent-forge
claude plugin install writing

# GitHub Copilot CLI
copilot plugin marketplace add tqnonline/agent-forge

# Codex CLI
codex plugin marketplace add tqnonline/agent-forge

# Amp
amp skill add github:tqnonline/agent-forge

# Gemini CLI
gemini skills install github:tqnonline/agent-forge

# Kilo Code, OpenCode, or Crush
pipx install tqn-agent-forge
agent-forge install writing --tier kilocode  # or opencode, crush
```

For Perplexity, ChatGPT GPTs, and Claude.ai Projects — see
[docs/install/_index.md](../install/_index.md).

## Compatibility

- Schemas frozen at v1.0; no breaking changes.
- Schemas/CLI surface/API are 100% compatible with v1.0.0. The only user-facing
  change is the PyPI install command (`tqn-agent-forge` instead of
  `agent-forge`). The `agent-forge` CLI command itself is unchanged.
- The `rahulnakmol/agent-forge` URL still redirects to `tqnonline/agent-forge`
  via GitHub's permanent rename redirects.
