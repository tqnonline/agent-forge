# agent-forge v1.1.0

**Released: 2026-05-26**

Minor release. Adds Factory.ai Droid as the 13th install target — covers both
the droid CLI and the Factory desktop app via the same canonical plugin layout.

## Added

- **Factory.ai Droid support** (Tier 1a — native marketplace registry). New
  translator emits `.factory-plugin/marketplace.json` at the repo root and
  per-plugin `.factory-plugin/plugin.json` files. Install:

  ```bash
  droid plugin marketplace add https://github.com/tqnonline/agent-forge
  droid plugin install writing@agent-forge
  ```

  Droid and the Factory desktop app share `~/.factory/` configuration, so a
  single translator covers both surfaces. Droid is documented as "compatible
  with plugins built for Claude Code," so the canonical plugin layout works
  without conversion. New install guide: `docs/install/factory-droid.md`.

## Install

```bash
pipx install tqn-agent-forge==1.1.0   # or: uv tool install tqn-agent-forge
```

Upgrade from v1.0.2:

```bash
pipx upgrade tqn-agent-forge
```

## Compatibility

- 100% backward compatible with v1.0.2. No schema or API changes.
- Adds a new value (`factory-droid`) to the `--tier` argument; existing tiers
  unchanged.
- Total install targets: 12 → 13.
